Formatações

Marcação

Podemos marcar uma palavra / frase usando a tag MARK.

Teste de uma nova Marcação de texto.


Texto grande e pequeno

Para criar um texto pequeno podera usar a tag SMALL.


Texto deletado

Podemos inserir a tag DEL, para marcar um texto que pode ser lido, porem não considerado.


Texto inserido

Podemos marcar um Texto inserido usando a tag INS.


Texto sobrescrito

Para inserir coisas do tipo x20+30 use a tag SUP.


Texto subscrito

Para inserir coisas do tipo H2O use a tag SUB.
